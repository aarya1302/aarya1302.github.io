Mental Load Application

Helping users reflect on and manage their mental load with Firebase, React Native Expo, and Qualtrics Integration.

Overview

The Mental Load app helps users assess and reflect on their mental labor. Key features include secure user authentication, interactive surveys, a visually appealing user interface, and real-time updates. The app combines Firebase for backend functionality, React Native Expo for cross-platform development, and Qualtrics for embedding custom surveys.

Key Features

Firebase Integration: Secure user authentication and real-time database management for dynamic updates.

React Native Expo: Cross-platform mobile development with a focus on clean and user-friendly design.

Qualtrics Surveys: Embedded reflective surveys to help users better understand their mental workload.

Technologies Used

React Native Expo
Firebase
Qualtrics
JavaScript
API Integration
Cross-Platform Development

Challenges and Solutions

Dynamic Survey Integration: Used React Native's WebView component and Firebase functions to fetch survey links dynamically, ensuring a seamless user experience.

Cross-Platform Compatibility: Thorough testing and adjustments ensured smooth performance on both iOS and Android devices.

Conclusion

The Mental Load app is a meaningful project that showcases my skills in full-stack development, third-party tool integration, and user-centric design. It helps users reflect on and manage their mental labor effectively.
